Esta página utiliza metadatos tipo META

En este fragmento se hace uso de los Tags meta con los atributos keywords y description, ambos elementos son empleados por los diversos "Search-Engines" en Internet para realizar una mejor clasificación acerca del contenido descrito en un documento. Vale mencionar que estos Tags no son obligatorios para que la página sea clasificada en un Motor de Búsqueda, sin embargo, si mejoran las posibilidades que el documento sea clasificado acordemente
