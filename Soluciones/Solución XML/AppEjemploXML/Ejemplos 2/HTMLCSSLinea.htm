Utilización de los CSS

Ejemplos de uso de los estilos

Definición CSS ("Cascading Style Sheets") es una metodología para dar uniformidad al uso de tags en documentos HTML / XHTML
Formas de Definir Estilos
  1. Uniformidad en Varios Documentos (Archivo CSS)
  2. Uniformidad en Documento (Tags <style> y <span> )
  3. Uniformidad en Tag ( Clases y en Linea )
